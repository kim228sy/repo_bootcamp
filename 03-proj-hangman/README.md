# HANGMAN GAME

## 사용 기술

- Yarn
- React
- Vite
- MUI
- Random-words

## 구현 요구 사항

- [✅] 랜덤 영단어를 글자 수에 맞게 화면 출력
- [✅] 입력값과 맞다면 화면에 출력 (중복 철자도 표시)
- [✅] 내가 입력한 단어 출력
- [✅] 10번 기회, 성공과 실패 구분

## 실행 방법

```
npm i -g yarn (yarn 설치 필요 시)
yarn install
yarn dev
```
